# Vite + React Template

This repository is a **project template** built with Vite, React and a predefined structure.

It is meant to be used as a **starting point for new projects**, not as a project itself.

---

## How to use this template

### Use GitHub Template

1. Open this repository on GitHub
2. Click **Use this template**
3. Choose:
   - Repository name
   - Owner (user or organization)
   - Public or private
4. Click **Create repository**

A new repository will be created with this codebase, **without affecting this template**.

---
